# Git 差异提取工具

一个基于 Python 的图形化工具，使用项目内置的 PortableGit 对比指定 Git 仓库的两个 Commit (SHA) 之间的差异，并将差异文件按原目录结构提取到指定输出目录中。

## 功能特性

- ✅ **Git 差异分析**: 对比两个 SHA 之间的文件变更
- ✅ **子模块支持**: 递归处理 Git Submodule 的差异
- ✅ **二进制文件安全**: 完整支持图片、编译文件等非文本文件
- ✅ **目录结构保持**: 维持原有的项目目录结构
- ✅ **配置记忆**: 自动保存用户输入，下次启动时恢复
- ✅ **进度显示**: 实时显示处理进度和日志
- ✅ **路径快捷操作**: 每个路径输入框都有浏览和打开按钮

## 系统要求

- Windows 操作系统
- Python 3.8+
- uv 包管理器
- 项目目录下需要包含 PortableGit 文件夹

## 安装与运行

### 快速启动
```bash
# 方法1: 使用启动脚本（推荐）
start.bat

# 方法2: 使用命令行
uv run python src/main.py
```

## 使用说明

### 基本操作
1. **选择仓库**: 点击"浏览"按钮选择 Git 仓库
2. **输入 SHA**: 输入要对比的两个提交 SHA 值
3. **设置输出**: 选择输出路径和文件夹名称
4. **预览差异**: 点击"List Diff"按钮查看差异文件列表
5. **执行提取**: 点击"Copy"按钮执行文件提取

### 输出结构
```
output_path/
└── folder_name/
    ├── old/          # 旧版本的文件
    └── new/          # 新版本的文件
```
